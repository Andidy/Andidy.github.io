finance-app is a project to create an application which allows a user to input financial events the app will then project those events into the future allowing the user to see how their wealth will grow or shrink over time.
It is also a project for me to refresh my C# knowledge, learn XAML and WPF, and famaliarize myself with new technologies, architectures, and design patterns.

The above image demonstrates the current functionality of finance-app. Including:

  • Creating finance events/objects
  • Creating finance balances
  • Calculating the value of balances based on a chosen date

Finance objects/events provide the following functionality:

  • A user provided name
  • An interval for potentially repeat occurences of an event
  • An amount the event affects the balance by
  • Whether the amount is a gain or loss to the balance
  • The date the event occurs on or the starting date (depending on interval type)
  • What balance the event will apply to

Finance Balances provide the following functionality:

  • A user provided name
  • A type (Savings, Checking, Credit Card, Stocks, etc.)
  • An amount describing the initial balance of the finance balance
